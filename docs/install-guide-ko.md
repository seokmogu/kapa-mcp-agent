# KAPA 에이전트 설치 안내 (KAPA가 설치된 PC에서)

이 PC에서 KAPA HUB PLUS / KAIS를 자동화하기 위한 작은 프로그램을 설치합니다.
**다운로드해서 더블클릭하면 끝**입니다. 추가 설치나 설정은 없습니다.

## 1. 프로그램 다운로드

아래 주소를 브라우저에 붙여넣어 파일 하나(`kapa-agent.exe`)를 받습니다.

> https://github.com/seokmogu/kapa-mcp-agent/releases/latest/download/kapa-agent.exe

## 2. 더블클릭으로 실행

받은 `kapa-agent.exe`를 더블클릭합니다.

- "Windows의 PC 보호" 같은 파란 경고창이 뜨면, **추가 정보**를 누른 뒤 **실행**을
  누릅니다. (이 프로그램은 아직 코드 서명이 없어서 처음 한 번 이렇게 뜹니다. 안전합니다.)
- 검은 콘솔창이 하나 뜨고 그대로 떠 있습니다. **이 창을 닫지 말고 그대로 두세요.**
  (이 창이 떠 있는 동안만 자동화가 동작합니다.)

> 동작 확인(선택): 브라우저에서 http://127.0.0.1:8765/health 를 열어
> `"ok":true` 가 보이면 정상입니다.

## 3. KAPA 실행 + 로그인

평소처럼 **KAPA HUB PLUS(또는 KAIS)를 실행하고, 공동인증서로 로그인**한 뒤,
작업할 화면(예: 주소/지번 검색 화면)까지 들어가 둡니다.

- 로그인과 인증서 입력은 **직접** 해주세요. 프로그램이 대신 누르지 않습니다.
- 로그인까지 끝난 상태를 이 에이전트가 이어받아 조회/추출을 돕습니다.

## 4. 준비 완료

여기까지 하면 끝입니다. 콘솔창은 계속 열어두고, "준비됐다"고 알려주세요.

---

### 참고

- 업데이트는 자동입니다. 다음에 기능이 추가되면 이 프로그램이 스스로 최신 버전을
  내려받아 교체합니다. 다시 설치할 필요 없습니다.
- 끄려면 콘솔창을 닫거나 창에서 Ctrl+C 를 누르면 됩니다.
- 인터넷은 GitHub로 나가는 연결(아웃바운드)만 사용합니다. 외부에서 이 PC로
  들어오는 연결은 열지 않습니다.
